AULA TEORICA

Monge Copista: quem copiava um livro tudo a mão

Johannes Gutenberg: invertor da prença mecanica de tipos (letras) móveis (como se fosse um carimbo de livro) século XV

Tipografia envolve os tipos das letras: FONTES

Anatomia do tipo.

ITENS MÉTRICOS

x: define a altura basica de todas as letras minúsculas. (altura x)

altura de vazamento por cima das minusculas: ASCENDENTE

Altura de vazamento pra baixo das minusculas: DESCENDENTE

Altura total das maiusculas: ALTURA DAS MAIUSCULAS

Soma total de todas as letras: ALTURA DO CORPO

ITENS ANATOMICOS (GEOMÉTRICOS)

Serifa: pezinho das fontes (curvinha no final) a serifa facilita a leitura pois cria uma linha imaginária.

Categoria das fontes

  • Serifadas
  • Sans-serif
  • Monoespaçadas
  • Handwriting (feito a mão)
